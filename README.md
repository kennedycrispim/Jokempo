
# Jogo -JOKEMPO
Desenvolvedor: Kennedy Crispim Gonçalves

GitHub: https://github.com/kennedycrispim
LinkedIn: www.linkedin.com/in/kennedy-crispim

O projeto JOKEMPO foi desenvolvido por meio da linguagem Javascript. A ideia surgiu apos uma aula da faculade onde após observar alguns projetos de surgiu-se a idea de desenvolver um game nostagico e interativo.
O projeto foi desenvolvido utilizando a IDE **VSCode**.


## Funcionalidades e detalhes
- O jogo é realizado contra a CPU(maquina)
- Apresenta uma caixa score, onde e possivel visualizar a pontuacao do jogo
- Botao de reiniciar, permite recomeçar o jogo.
